Estrutura Básica do HTML

(Voltar)

A estrutura básica do HTML possui as tags HTML, Header e Body. O Header é carregado antes de apresentar o body. O Body representa o que será apresentado ao usuário.

Tag

(Voltar)

As Tags normalmente são fechadas, porém, existem algumas que não são, por exemplo, as tags input e img.

Atributo

(Voltar)

Os atributos são propriedades das tags. Existem atributos mais genéricos e outros específicos a determinadas tags. Podemos citar atributos genéricos o id, style e class e como especificos: tag input, o type que pode assumir alguns valores como: text, number e color. tag img, width e src.

Tipografia

(Voltar)

Composta por títulos e subtítulos correspondentes às tags h1 a h6. Sendo que cada h possui sua formatação específica por navegador. Parágrafos, tag p. Citações, tag blockquote. Outras tags como sublinhado u, itálico i, negrito strong, marca texto mark e posicionamento acima da altura normal sup.

Listas ordenadas e não ordenadas

(Voltar)

Listas ordenadas utilizar a tag lo e as não ordenadas li. Para cada item, utilizar a tag li. A lista ordenada possui numeração enquanto a não ordenada é itemizada por símbolos. As listas podem ser um item.

Âncora

(Voltar)

Tag a com atributo href na qual pode-se informar um endereço da internet ou do próprio projeto. atributo target utilizar o valor _self quando se deseja abrir na mesma aba do navegador e _blank quando se deseja abrir em outra página do navegador. Pode-se criar uma ancoragem saltando de um lugar par outro na mesma página. Basta criar o id para onde se deseja saltar e utilizar no href o id precedido do caracter #.
